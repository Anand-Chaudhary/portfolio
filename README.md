## Getting Started

First, clone the repository and install dependencies:

```bash
git clone <your-repo-url>
cd portfolio
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
